# AWS S3 Static Website Hosting

This project demonstrates how to host a simple static website using Amazon S3.

## Architecture
- Amazon S3 bucket for hosting static website
- Public read bucket policy
- Static website hosting enabled

## Steps implemented
1. Created an S3 bucket
2. Uploaded website files
3. Enabled static website hosting
4. Configured bucket policy for public access
5. Verified website endpoint
